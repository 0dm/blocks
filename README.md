# Blocks

Very small minecraft clone for macOS.

The current build comes out to about 35 KB and runs as a native Cocoa app with no external libraries.

## Build

```sh
make
```

## Run

```sh
make run
```

## Controls

- Click to capture the mouse
- `WASD` move
- `Space` jump
- `Shift` sprint
- Mouse look
- Left click break block
- Right click place block
- `1` `2` `3` `4` choose block type
- `Esc` release mouse, then `Esc` again quits
